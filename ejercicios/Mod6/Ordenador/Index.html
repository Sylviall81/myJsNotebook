M.8.2 Ej.1. Programa para crear Ordenadores

Ejercicio 6.3.Instrucciones:

Crea una clase en JavaScript llamada "Ordenador", en un archivo llamado ordenador.js. Esta clase tendrá los siguientes
atributos:

  • marca
  • modelo
  • procesador
  • Memoria RAM
  • capacidad del disco duro
Además del constructor, debe disponer de los siguientes
métodos:
  • Métodos que permitan consultar cada uno de los atributos (getters).
  • Métodos que permitan modificar el procesador, la cantidad de memoria RAM y la capacidad del disco duro. (setters).
  • Un método que recibirá un parámetro y devolverá una frase que diga: "En estos momentos se está ejecutando: 'X'" (X es el parámetro recibido por el método y que normalmente será el nombre de un programa). Ejemplo: "En estos momentos se está ejecutando: Chrome"
  • Un método que devolverá una descripción completa del ordenador (toString()).
En otro archivo llamado main.js, crea la función crear Ordenadores() . Dentro de esta función, pediremos al usuario mediante promtps los datos de dos ordenadores. Una vez introducidos los datos de ambos ordenadores, llama a los métodos de la clase Ordenador para comprobar que funcionan correctamente.
