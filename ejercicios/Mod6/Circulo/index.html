M.8.2| Ej.4 Programa para crear círculos

Ej. 4 Instrucciones:

Hacer una aplicación en la que el usuario introduce las coordenadas X, Y de dos círculos y sus respectivos radios.

A.Crear la clase Punto y la clase Círculo:

  • Clase Punto:
    Tendrá los ATRIBUTOS: posicion X y posicion Y. Además tendrá el constructor parametrizado con todos los atributos de la clase.
  • Clase Círculo:
    Tendrá los ATRIBUTOS: centro (un punto) y el radio (distancia entre el centro y cualquier punto de la circunferencia). Además, tendrá el constructor con todos los parámetros y los siguientes siete (7) métodos:
  1. DistanciaCentros(otroCírculo) : Devuelve la distancia entre el centro del círculo actual y el recibido como parámetro.
    Fórmula de distancia entre dos puntos.
  2. Equal(otroCírculo) : Devuelve true si dos círculos son iguales (si tienen el mismo centro y el mismo radio).
  3. sonConcentricos(otroCírculo) : Devuelve true si dos círculos son concéntricos (tienen igual centro).
  4. tienenIgualRadio(otroCírculo) : Retorna true si dos círculos tienen el mismo radio.
  5. sonTangentes(otroCírculo) : Devuelve true si dos círculos son tangentes.
  6. sonSecantes(otroCírculo) : Devuelve true si dos círculos son secantes.
  7. noSeTocan(otroCírculo) : Devuelve true si dos círculos no se tocan.
B. Finalmente, la aplicación mostrará por pantalla el resultado de realizar la llamada a todos los métodos de la clase Círculo.

podemos obtener la siguiente información:

  1. Distancia entre los centros:

    Método: fórmula de distancia entre dos puntos.

  2. ¿Son dos círculos iguales?

    Dos círculos son iguales si tienen el mismo centro y el mismo radio.

  3. ¿Son concentricos?

    Dos circulos son CONCENTRICOS si tienen igual centro.

  4. ¿Tienen el mismo Radio(r)?

  5. ¿Son Tangentes?

    Dos círculos son TANGENTES (sólo se tocan en un sólo punto) si la distancia entre sus centros es igual a la suma de sus radios

  6. ¿Son Secantes?

    Dos círculos son SECANTES (se tocan en dos puntos) cuando la distancia entre sus centros es menor que la suma de sus radios*.

  7. ¿Los círculos se tocan?

    8. Dos círculos NO SE TOCAN si la distancia entre sus centros es mayor a la suma de sus radios*.

    *Nota: Si las circunferencias son concéntricas NO SE TOCAN (y por tanto, tampoco pueden ser secantes)
    aún cuando la distancia entre sus centros (0) sea menor que la suma de sus radios
