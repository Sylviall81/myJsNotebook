(M8.2) Ejercicio 3. Programa para crear cuentas de clientes.

Ej.3 Instrucciones:

Dado el diagrama de clases de la siguiente imagen:

Programa una aplicación que tenga 5 botones:

  1. Crear cliente/a.
  2. Eliminar cliente/a.
  3. Crear cuenta de un cliente/a.
  4. Ingresar euros en una cuenta de un cliente/a.
  5. Retirar euros en una cuenta de un cliente/a.

Info adicional:
  • El saldo inicial de todas las cuentas debe ser 0€.
  • Al crear un cliente, no tendrá que tener ninguna cuenta “asociada”.
  • El diagrama UML muestra las propiedades y métodos mínimos que deben tener las clases, pero puedes crear más si es necesario.
